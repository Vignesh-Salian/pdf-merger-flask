# 📄 PDF Merger Web App

A simple web application built using **Flask** that allows users to upload multiple PDF files and merge them into a single downloadable PDF.

---

## 🚀 Features

* Upload multiple PDF files
* Merge them into one PDF
* Download the merged PDF instantly
* Simple and clean user interface

---

## 🛠️ Tech Stack

* Python
* Flask
* PyPDF2
* HTML
* CSS

---

## ▶️ How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Vignesh-Salian/pdf-merger-flask.git
```

### 2️⃣ Navigate to the project folder

```bash
cd pdf-merger-flask
```

### 3️⃣ Install required dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
python app.py
```

### 5️⃣ Open in your browser

```
http://127.0.0.1:5000
```
